## 🚀 The Cosmic Experience

### 🎭 Choose Your Fate
- **Professional Mode**: Get an expert recruiter's-eye view. Strengths, weaknesses, and a roadmap to your next promotion.
- **Savage Roast**: From "Light Heat" to "Absolute Zero," watch as the AI dismantles your buzzwords and over-inflated job titles.
- **The Both-Sided Coin**: Get the best of both worlds. The truth you need and the roast you deserve.

### 📡 High-Frequency Features
- **PDF & DOCX Support**: Simply drop your resume and let the AI parse the chaos.
- **LinkedIn Integration**: Paste your URL and see how your public persona holds up.
- **Skill Gap Engine**: Automatically identifies what's missing between you and your dream role.
- **The "Fix It" Button**: One click to have the AI rewrite your vaguest bullet points into metric-driven wins.
- **Meteor Loading UI**: Forget boring spinners. Experience a high-intensity meteor shower while the AI computes your doom.

## 🛸 Zero-G Setup

### 1. Clone the Galaxy
```bash
git clone https://github.com/litgrxpher/HireMeOrRoastMe.git
cd HireMeOrRoastMe
```

### 2. Fuel the Engines (Environment Variables)
Create a `.env` in the root directory:
```env
GEMINI_API_KEY=your_gemini_key_here
VITE_API_BASE_URL=/api
```

### 3. Launch Sequences
```bash
# Install root dependencies
npm install

# Run Frontend (Vite)
cd frontend
npm install
npm run dev
```
